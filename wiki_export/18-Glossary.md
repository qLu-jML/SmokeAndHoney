# Glossary


## Glossary


| AFB | American Foulbrood. A highly contagious bacterial brood disease. Notifiable in most regions. |
| --- | --- |
| Apiary | The physical site where beehives are kept. A beekeeper may manage multiple apiaries. |
| Bearding | Cluster of bees hanging outside the hive entrance, usually due to heat or overcrowding. |
| Brood | Developing bee eggs, larvae, and pupae inside the hive. Capped brood is sealed with wax. |
| Brood Pattern | The distribution of capped brood on a frame. Solid pattern = healthy queen. Spotty = problem. |
| Buckfast | A hybrid bee developed at Buckfast Abbey. Disease-resistant, calm, reliable production. |
| Candy Plug | Fondant used to slow-release a new queen from her introduction cage. |
| Carniolan | Apis mellifera carnica. Gentle, strong spring buildup, frugal in winter. Swarm-prone. |
| Caucasian | Apis mellifera caucasica. Long tongue, good forager. Slow spring buildup. |
| Chalkbrood | Fungal disease turning bee larvae into hard, chalky white mummies. |
| DCA | Drone Congregation Area. Airspace where drones from many colonies gather to mate with virgin queens. |
| Dandelion Flow | The spring nectar flow from wild dandelions. A stochastic annual event — density, timing, and duration vary year to year. Make-or-break for spring colony survival in lean years. Dandelion honey is a spring specialty with above-average market value. |
| Deformed Wing Virus (DWV) | Virus transmitted by Varroa mites. Causes stunted, crumpled wings in adult bees. |
| Drone Layer | A queen laying only unfertilized eggs — produces only drones. Colony will collapse without intervention. |
| EFB | European Foulbrood. A less severe brood disease caused by bacteria. Responds to improved nutrition. |
| Flow (Nectar Flow) | A period of abundant nectar availability from blooming plants. Bees produce surplus honey during a flow. |
| Goldenrod Flow | The fall nectar flow from wild goldenrod. A stochastic annual event — the last major flow of the Cedar Bend beekeeping year. Critical for winter store building. Goldenrod honey is dark, strongly flavored, and sells poorly at market. Can be terminated early by frost. |
| Foundation | Wax or plastic sheet embossed with a hexagonal pattern, installed in frames to guide comb construction. |
| Frame | Removable wooden structure inside a hive box on which bees build comb. |
| Italian | Apis mellifera ligustica. Most popular beginner bee. Gentle, high honey production, excellent brood pattern. |
| K-wing | Characteristic wing posture of bees infected with tracheal mites. |
| Langstroth | The most common modern hive design, featuring removable frames and standardized box sizes. |
| Laying Worker | A worker bee that begins laying unfertilized eggs in a queenless colony. Very difficult to correct. |
| Linden | Also called basswood (Tilia spp.). One of the most prized honey trees, producing a premium light honey. |
| Mead | Fermented beverage made from honey and water, often with added fruits or spices. |
| Mouse Guard | A metal entrance strip that prevents mice from entering hives in winter. |
| Nectar Dearth | A period when little or no nectar is available. Can cause robbing and colony stress. |
| Nosema | A gut fungal infection (Nosema apis or ceranae). Causes dysentery and weak spring buildup. |
| Nuc (Nucleus Colony) | A small established colony on 4–5 frames with a mated, laying queen. Used to start or boost a hive. |
| Package | A screened box of bees (~3 lbs, ~10,000 bees) with a caged, mated queen. Ordered to start new hives. |
| Pollen Substitute | A protein supplement fed to bees when natural pollen is scarce, especially in early spring. |
| Queen | The single reproductive female in a colony. Responsible for all fertilized egg production. |
| Queen Cell | A large, peanut-shaped wax cell in which a new queen is raised. |
| Refractometer | A tool that measures the moisture content of honey. Below 18.5% is acceptable; below 17% is premium. |
| Requeen | To replace the existing queen with a new one, usually to improve genetics or colony behavior. |
| Robbing | When bees from one colony raid the honey stores of a weaker neighboring colony. |
| Russian | A Varroa-resistant bee strain developed from Far Eastern Apis mellifera. |
| Sacbrood | A viral brood disease. Affected larvae fail to pupate and resemble a fluid-filled sac. |
| Small Hive Beetle (SHB) | Aethina tumida. A scavenging beetle that can damage comb and honey if populations go unchecked. |
| Smoker | A tool that burns natural fuel to produce cool smoke, which calms bees during inspections. |
| Split | Dividing a colony into two to prevent swarming or to expand the apiary. |
| Sugar Roll | A mite-testing technique: bees are shaken with powdered sugar to dislodge mites for counting. |
| Super | A shallower hive box added above the brood nest specifically for honey storage. |
| Supersedure | The colony's own process of replacing an old or failing queen with a newly raised one. |
| Swarm | Natural colony reproduction: the old queen leaves with half the bees to form a new colony elsewhere. |
| Tracheal Mites | Acarapis woodi. Internal mites that infest bee respiratory tracts. Identified by K-wing. |
| Varroa | Varroa destructor. The single most destructive honeybee parasite worldwide. |
| Varietal Honey | Honey produced predominantly from a single plant source (e.g., clover honey, linden honey). |
| Warré Hive | A vertical top-bar hive designed for minimal intervention beekeeping. |
| Wax Moth | Galleria mellonella. Larvae destroy comb in weak or dead hives. |
| Winter Bees | Long-lived bees raised in late summer / fall with enlarged fat bodies to survive winter and raise spring brood. |
| Worker | An infertile female bee. Performs all colony tasks: foraging, nursing, guarding, building, heating. |
