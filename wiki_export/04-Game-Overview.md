# Game Overview



| Title | Smoke & Honey |
| --- | --- |
| Genre | Simulation, Strategy |
| Platform | PC (primary), Mobile (secondary) |
| Target Audience | Ages 12+; simulation enthusiasts, casual gamers, nature/agriculture hobbyists |
| Tone | Warm, grounded, educational — with real seasonal tension and authentic depth |
| Session Length | PC: 30–90 minutes. Mobile: 10–20 minutes. Game is designed for both. |
| Save Structure | Auto-save at end of each in-game week. Manual save at any time. |


### 4.1 Design Pillars


| Authentic Simulation | Every mechanic is grounded in real beekeeping. Players who learn the game learn real beekeeping principles. |
| --- | --- |
| Observation Over Stats | The player reads their bees, not a dashboard. Information is earned through inspection and experience. |
| Consequential Seasons | Decisions in one season carry real consequences into the next. Fall preparation determines winter survival. |
| Relaxed Pressure | The game is challenging but never punishing. A lost hive is a setback, not a game-over. The player always has a path forward. |
| Rewarding Mastery | A player who understands bee biology will have visibly better outcomes than one who guesses. Knowledge matters. |
