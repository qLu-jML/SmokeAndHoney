====================================

Quick reference showing all major systems, their key mechanics, and parameter ranges.

-------------------|--------------------------------------------------|------------------
Queen Grade        | S/A/B/C/D/F, affects laying rate               | +25% (S) to -100% (F)

Brood Pattern      | Visible on frames, indicates queen health       | Solid/Spotty/Failing
Frame Capacity     | Cell grid for deep/super frames                 | Deep: 3,500 cells, Super: 2,450 cells
Nectar/Honey       | Cells filled by foragers, stored for consumption| Visible as amber fill on frames
Pollen Storage     | Colored cells (orange/yellow) around brood      | Measured as % of frame area
Equipment Condition| Degrades weekly, affects production/pest risk   | 0-100 scale, -1 to -4 per week
Colony Health Score| Hidden composite metric                         | 0-100, shown as color state
Stress Multiplier  | Affects laying rate based on conditions         | ×1.0 baseline to ×0.40 (very stressed)

SYSTEM: STRESS MODIFIER FORMULA
================================================================================

Base Laying Rate Calculation:
final_rate = base × grade_mod × age_mult × stress_mult × season_mod

Stress Multiplier Components:
Brood-bound (>90% full):           ×1.15
No food stores (critical):         ×1.20
Population low:                    ×1.10
Disease/pest load:                 ×1.25
Queenless:                         ×1.50
Inspection >2/week:                ×1.15
Strong nectar flow:                ×0.85
Adequate space (not congested):    ×0.95
Healthy young queen:               ×0.90

Example: Healthy hive with good forage = 1.15 × 0.95 × 0.90 = 0.98x (minor positive)
Example: Stressed hive with disease = 1.25 × 1.15 = 1.44x (44% reduction in actual laying)

SYSTEM: SAVE/LOAD
================================================================================

Persistent Data:
- All hive state (queen, population, frames, equipment condition)
- Player progress (XP, standing, inventory, cash)
- Calendar/time (current season, day, time of day)
- Weather state (current and weekly forecast)
- NPC relationship status and quest progress
- Apiary structures unlocked
- Game settings and player preferences

Serialization Method:
- JSON or binary format (implementation detail)
- Automatic save on day transition
- Manual save available in pause menu
- Multiple save slots supported (at least 3)
- Save/load timing: O(n) where n = hive count + NPC count (should be <1 second)

Data Integrity:
- Checksums prevent save file tampering
- Backup on save (keep previous version)
- Graceful handling of corrupted saves
