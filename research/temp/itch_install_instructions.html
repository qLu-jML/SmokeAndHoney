How to Download & Install

1. Click the Download button above to get the .zip file.

2. Right-click the .zip and select Extract All (or use 7-Zip/WinRAR). Do not run the game from inside the zip — it won't work.

3. Open the extracted folder. You should see SmokeAndHoney.exe and SmokeAndHoney.pck — both files must stay in the same folder.

4. Double-click SmokeAndHoney.exe to play.

Windows SmartScreen Warning

Windows may show a "Windows protected your PC" popup because the game isn't signed yet. This is normal for indie games. Click More info, then click Run anyway to launch the game.

Requirements

OS: Windows 10 or later (64-bit)

Storage: ~200 MB

No install required. Just extract and play.
